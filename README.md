# Python Fundamentals Assessment

## Overview
This is a **2 hour Python fundamentals test** designed for learning core programming concepts.

It tests your ability to:
- Write and call functions
- Use conditionals, loops, and lists
- Handle strings and numbers
- Solve simple logic problems

There are **12 problems total**

## Instructions
1. All tasks are in `fundamentals.py`.
2. Open and complete each function where `pass` is written.
3. To run tests, execute:
   ```bash
   python -m unittest tests/test_fundamentals.py
   ```
4. Focus on correctness and clarity — simple code wins.
